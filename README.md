# goods
用vue做的电商项目
 //启动项目  
 
 npm install  
 
 npm run mock  
 
 npm run dev  
 

 //使用vuex实现多组件之间数据通信，

   vue-lazyload实现图片懒加载，  
   
   json-server插件实现mock数据
